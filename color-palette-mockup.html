Typography Examples

See how text looks with the color palette

Heading Level 1

This is a paragraph demonstrating body text in Rich Charcoal (#1A1A1A) on a Soft Cream (#F5F1E8) background. The contrast ratio of 14.8:1 ensures excellent readability for all users.

Heading Level 2

All headings use the Deep Merlot color (#6B1C23) to create visual hierarchy and draw attention to important sections. This creates a sophisticated and elegant look.

Heading Level 3

Links are styled in the accent color and include a subtle underline. Here's an example link that demonstrates the hover effect when you move your cursor over it.

The typography system uses Cormorant Garamond for headings (elegant serif) and Montserrat for body text (clean sans-serif), creating a perfect balance between sophistication and readability.
