## Introduction

This style guide defines the color palette for HCJK Photography's brand identity. The palette combines three sophisticated colors—Beige, Black, and Merlot—to create a warm, elegant, and professional aesthetic that reflects the luxury and artistry of fine photography.

### Design Philosophy

- **Warmth:** Beige tones create an inviting, comfortable atmosphere
- **Sophistication:** Merlot adds luxury and visual interest
- **Clarity:** Black provides excellent readability and contrast
- **Accessibility:** All combinations meet WCAG AAA standards

## Primary Palette

These three colors form the foundation of the HCJK brand identity.

### 1. Soft Cream
**Hex:** `#F5F1E8`  
**RGB:** `rgb(245, 241, 232)`  
**HSL:** `hsl(40, 36%, 94%)`

**Primary Uses:**
- Main page backgrounds
- Card backgrounds
- Section backgrounds
- Email backgrounds

**Characteristics:**
- Warmest beige option
- Reduces eye strain compared to pure white
- Creates inviting atmosphere
- Works as primary background color

**Contrast Ratios:**
- With Rich Charcoal: **14.8:1** (AAA ✓)
- With Deep Merlot: **10.2:1** (AAA ✓)
- With Pure Black: **19.6:1** (AAA ✓)

---

### 2. Rich Charcoal
**Hex:** `#1A1A1A`  
**RGB:** `rgb(26, 26, 26)`  
**HSL:** `hsl(0, 0%, 10%)`

**Primary Uses:**
- Body text
- Paragraphs
- Navigation links
- Form labels
- Footer text

**Characteristics:**
- Softer than pure black
- Easier on eyes for extended reading
- Maintains excellent readability
- Professional and authoritative

**Contrast Ratios:**
- With Soft Cream: **14.8:1** (AAA ✓)
- With Warm Sand: **12.5:1** (AAA ✓)
- With Golden Beige: **9.8:1** (AAA ✓)

---

### 3. Deep Merlot
**Hex:** `#6B1C23`  
**RGB:** `rgb(107, 28, 35)`  
**HSL:** `hsl(355, 59%, 26%)`

**Primary Uses:**
- Primary buttons
- Call-to-action elements
- Main headings (H1, H2)
- Links and hover states
- Important highlights
- Brand accents

**Characteristics:**
- Rich, sophisticated burgundy
- Commands attention without being aggressive
- Conveys luxury and elegance
- Creates strong visual hierarchy

**Contrast Ratios:**
- With Soft Cream: **10.2:1** (AAA ✓)
- With Warm Sand: **8.6:1** (AAA ✓)
- With Golden Beige: **6.8:1** (AA ✓)

## Color Usage Guidelines

### Background Colors

#### Primary Background
**Color:** Soft Cream (#F5F1E8)
- Use for main page backgrounds
- Provides warm, inviting base
- Reduces eye strain
- Creates cohesive look

#### Alternate Background
**Color:** Warm Sand (#E8DCC8)
- Use for alternating sections
- Creates visual separation
- Adds depth to layout
- Maintains warmth

#### Dark Background
**Color:** Rich Charcoal (#1A1A1A)
- Use for footer
- Hero overlays
- Contrast sections
- Premium feel areas

---

### Text Colors

#### Primary Text
**Color:** Rich Charcoal (#1A1A1A)
**Usage:**
- Body paragraphs
- Navigation links
- Form labels
- General content

**Best Practices:**
- Always use on light backgrounds
- Minimum font size: 16px
- Line height: 1.6-1.8
- Never use on dark backgrounds

#### Heading Text
**Color:** Deep Merlot (#6B1C23)
**Usage:**
- H1, H2 headings
- Section titles
- Important callouts
- Brand messaging

**Best Practices:**
- Use for visual hierarchy
- Pair with Rich Charcoal for subheadings
- Maintain consistent sizing
- Don't overuse

#### Light Text
**Color:** Soft Cream (#F5F1E8)
**Usage:**
- Text on dark backgrounds
- Footer content
- Hero section text
- Overlay text

**Best Practices:**
- Only use on dark backgrounds
- Ensure sufficient contrast
- Test readability
- Consider font weight

---

#### Primary Accent

## Accessibility & Contrast

### WCAG Compliance

All color combinations in this palette meet or exceed WCAG 2.1 Level AAA standards.

#### Contrast Ratio Requirements

| Text Size | WCAG AA | WCAG AAA | Our Standard |
|-----------|---------|----------|--------------|
| Normal Text (< 18px) | 4.5:1 | 7:1 | **10.2:1+** |
| Large Text (≥ 18px) | 3:1 | 4.5:1 | **10.2:1+** |

### Approved Color Combinations

#### Excellent Contrast (AAA)

| Background | Text Color | Contrast | Rating |
|------------|-----------|----------|--------|
| Soft Cream | Rich Charcoal | 14.8:1 | ★★★★★ |
| Soft Cream | Deep Merlot | 10.2:1 | ★★★★★ |
| Soft Cream | Pure Black | 19.6:1 | ★★★★★ |
| Rich Charcoal | Soft Cream | 14.8:1 | ★★★★★ |
| Deep Merlot | Soft Cream | 10.2:1 | ★★★★★ |

#### Good Contrast (AA)

| Background | Text Color | Contrast | Rating |
|------------|-----------|----------|--------|
| Warm Sand | Rich Charcoal | 12.5:1 | ★★★★☆ |
| Warm Sand | Deep Merlot | 8.6:1 | ★★★★☆ |
| Golden Beige | Rich Charcoal | 9.8:1 | ★★★★☆ |

## Do's and Don'ts

### ✅ DO

1. **Use Soft Cream for large background areas**
   - Creates warmth and reduces eye strain
   - Provides excellent base for content

2. **Use Deep Merlot sparingly for maximum impact**
   - Reserve for CTAs and important elements
   - Maintains visual hierarchy

3. **Use Rich Charcoal for all body text**
   - Ensures optimal readability
   - Softer than pure black

4. **Test all color combinations for accessibility**
   - Use contrast checkers
   - Verify on different devices

5. **Maintain consistent color usage across pages**
   - Builds brand recognition
   - Creates cohesive experience

6. **Use hover states to indicate interactivity**
   - Classic Burgundy for button hovers
   - Provides visual feedback

7. **Add subtle beige variations for depth**
   - Warm Sand for alternate sections
   - Golden Beige for borders

8. **Use gradients for hero sections**
   - Merlot to Burgundy gradient
   - Creates visual interest

---

### ❌ DON'T

1. **Don't use Pure Black (#000000) for body text**
   - Too harsh for extended reading
   - Use Rich Charcoal instead

2. **Don't overuse merlot**
   - Should be accent, not dominant
   - Loses impact if overused

3. **Don't place merlot text on beige without testing**
   - Some combinations may not meet contrast standards
   - Always verify accessibility

4. **Don't mix all three merlot variations in one section**
   - Creates visual confusion
   - Stick to one or two variations

5. **Don't use beige text on beige backgrounds**
   - Obvious but worth noting
   - Ensure sufficient contrast

6. **Don't ignore mobile color rendering**
   - Colors may appear different on mobile
   - Test on actual devices

7. **Don't use colors inconsistently**
   - Maintain button color standards
   - Keep link colors uniform

8. **Don't forget about print**
   - Some colors may not print well
   - Test print versions

## Implementation Guide

### Step 1: Set Up CSS Variables

Add this to your main CSS file:

```css
:root {
  /* Primary Palette */
  --color-primary-bg: #F5F1E8;
  --color-primary-text: #1A1A1A;
  --color-primary-accent: #6B1C23;
  
  /* Beige Variations */
  --beige-light: #F5F1E8;
  --beige-medium: #E8DCC8;
  --beige-dark: #D4C4A8;
  
  /* Black Variations */
  --black-pure: #000000;
  --black-rich: #1A1A1A;
  --black-slate: #2C2C2C;
  
  /* Merlot Variations */
  --merlot-deep: #6B1C23;
  --merlot-classic: #8B2332;
  --merlot-rich: #A52A3A;
}
```

### Step 2: Apply Base Styles

```css
body {
  background-color: var(--color-primary-bg);
  color: var(--color-primary-text);
  font-family: 'Montserrat', sans-serif;
}

h1, h2, h3 {
  color: var(--color-primary-accent);
  font-family: 'Cormorant Garamond', serif;
}

a {
  color: var(--color-primary-accent);
}

a:hover {
  color: var(--merlot-classic);
}
```

### Step 3: Create Component Styles

```css
.btn-primary {
  background-color: var(--merlot-deep);
  color: var(--beige-light);
}

.btn-primary:hover {
  background-color: var(--merlot-classic);
}

.card {
  background-color: var(--beige-light);
  border: 1px solid var(--beige-dark);
}
```

### Step 4: Test Accessibility

1. Use WebAIM Contrast Checker
2. Test with screen readers
3. Verify on different devices
4. Check color blindness simulation

### Step 5: Document Usage

- Create component library
- Document color decisions
- Maintain style guide
- Train team members
