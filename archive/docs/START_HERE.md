### Action 1: View Your Website (5 minutes)
```bash
# Option A: If you have Python installed
python -m http.server 8000
# Then open: http://localhost:8000

# Option B: Use VSCode Live Server
# 1. Open index.html in VSCode
# 2. Right-click â†’ "Open with Live Server"
```

### Action 2: Customize Colors (10 minutes)
Open `assets/css/variables.css` and change:
```css
:root {
  --color-primary: #2C2C2C;      /* Your main color */
  --color-secondary: #8B7355;    /* Your accent color */
  --color-accent: #D4AF87;       /* Your highlight color */
}
```

### Action 3: Add Your Logo (15 minutes)
1. Export your logo as SVG or PNG
2. Save to `assets/images/logo/logo.svg`
3. Update `index.html` line 73:
```html
<img src="assets/images/logo/logo.svg" alt="HCJK Collection Logo">
```
