# HCJK Collection Website Poetry Transformation

## Overview
This document describes the complete transformation of the HCJK Collection website from a photography portfolio to a poetry and mental health platform.

## Changes Made

### New Pages Created
1. **book.html** - Dedicated book page for "I See You; I See Me"
2. **poems.html** - Poetry showcase organized by themes
3. **resources.html** - Mental health resources and crisis support

### Pages Completely Transformed
1. **index.html** - Transformed from photography homepage to poetry platform
2. **about.html** - Rewritten as author biography and mission statement
3. **contact.html** - Rebuilt with contact form, FAQ, and crisis resources

### New CSS Files
1. **assets/css/poetry-styles.css** - Complete poetry-focused styling
2. **assets/css/contact-styles.css** - Contact page specific styles

### New JavaScript Files
1. **assets/js/poetry-interactions.js** - Interactive features for poetry pages
2. **assets/js/contact-form.js** - Contact form handling and FAQ accordion

### Pages to be Removed/Archived

## Design Philosophy

### Color Palette
- **Primary Blue**: #1a365d - Depth, introspection
- **Secondary Blue**: #2c5282 - Trust, stability
- **Accent Gold**: #d69e2e - Hope, light
- **Light Cream**: #faf7f2 - Softness, readability
- **Dark Text**: #2d3748 - Readability
- **Light Text**: #718096 - Subtle text

### Typography
- **Headings**: Georgia serif - Timeless, literary feel
- **Body**: System fonts - Readability
- **Poetry**: Georgia, larger sizes, generous line spacing
